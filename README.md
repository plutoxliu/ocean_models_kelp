# Ocean_models_kelp
SDMs and particle dispersal modelling for giant kelp in the Southern Ocean
<br>
There are three folders containing:
- Species Distribution Model using R package "embarcadero" https://github.com/cjcarlson/embarcadero
- Lagrangian oceanogeaphic dispersal model using python package "Parcels" https://parcels-code.org/#projects
- Oceanographic model using "Regional Ocean Modeling System (ROMS)" https://www.myroms.org/
  <br>(note the ROMS model is under construction, but the folder contains codes to generate files needed to run ROMS, details in README file in the folder)

All models are optimised to focus on the circumpolar Southern Ocean and Antarctic regions
